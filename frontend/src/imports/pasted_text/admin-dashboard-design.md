## 1. デスクトップ管理画面設計方針

### 1.1 ターゲットデバイス・環境

```
画面サイズ: 768px〜1280px（タブレット〜ワイドデスクトップ）
基準フレーム: 1280×900px
向き: 横固定
動作環境: Webブラウザ（Chrome/Edge/Safari推奨）
使用時間帯: 営業時間内（8:00〜18:00）
操作デバイス: マウス＋キーボード（ショートカット対応）
```

### 1.2 対象ユーザー

| ユーザー | 役割 | UI最適化 |
|---------|------|----------|
| 店長 | 全体管理・KPI確認 | ダッシュボード・レポート重視 |
| 区域管理者 | 担当区域の配達管理 | フィルタリング・区域ビュー |
| 事務スタッフ | 購読者・留守止めデータ入力 | フォーム効率・一括操作 |

### 1.3 共通レイアウト構造

```
┌────────┬──────────────────────────┐
│        │      トップバー (64px)     │
│  左     ├──────────────────────────┤
│  サイド  │                          │

│ (240px) │     (1040px)              │

```

```

## 4. デスクトップ固有インタラクション

### 4.1 データテーブル操作

```
ソート: カラムヘッダークリックで昇順/降順切替（矢印アイコン表示）
フィルター: ヘッダー右のファネルアイコンでインラインフィルター
一括選択: ☐ヘッダーチェックで全選択、一括操作バー表示
インライン編集: セルダブルクリックで直接編集モード（対応列のみ）
行アクション: ⋯メニュー or ホバー時に操作アイコン表示
キーボード:
  ↑↓ で行移動、Enter で詳細、Delete で削除確認
```

### 4.2 ドラッグ&ドロップ

```
ルート並び替え: ☰ハンドルをドラッグ → ドロップ位置にスナップガイド表示
シフトカレンダー: ブロックをドラッグ → 日付/時間セルにスナップ
区域境界: ポリゴン頂点をドラッグ → スナップグリッド
ドラッグ中: 半透明のゴースト要素 + 挿入位置にブルーライン
```

### 4.3 モーダル・ドロワー

```
モーダル:
  - 表示: フェードイン（250ms）+ スケール0.95→1.0
  - 閉じる: Escキー or 背景クリック or ×ボタン
  - 背景: #1A1A1A opacity-50

ドロワー（右パネル）:
  - 幅: 320px〜480px
  - 表示: 右からスライドイン（250ms）
  - 閉じる: Esc or ×
  - メインコンテンツはpush（縮小）
