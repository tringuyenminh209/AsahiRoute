# AsahiRoute

AsahiRoute (朝日ルート) は、朝日新聞の販売店（ASA）に特化した、クラウドベースの次世代「新聞配達・ルート最適化システム」です。
配達員向けの直感的な「モバイルアプリ」と、店長・管理者向けの強力な「デスクトップダッシュボード」を統合し、配達業務の効率化と人的ミスの削減を実現します。

---

## プロジェクト開発の目的と目標 (Project Objectives)

近年の新聞配達業界において、配達員の人手不足、高齢化、ならびに外国人留学生などの多様な人材への依存が増加しています。これに伴い、「複雑な配達ルートの記憶」「日々の変更（留守止めや新規）の共有ミス」といった属人的な業務課題が浮き彫りになっています。

本プロジェクトの主たる目的は以下の通りです：

1. **属人化からの脱却**: 熟練スタッフの頭の中にあるルート情報をデジタル化し、新人もしくは代走スタッフでも即日配達可能な環境を構築する。
2. **AIによるルート最適化**: 日々の購読者の変動（新規追加、留守止めなど）をシステムが即座に分析し、最も効率的な配達順序・経路を自動生成する。
3. **ユニバーサルデザインの実現**: 高齢者でも見やすい十分なコントラスト・サイズ設定に加え、日本語の読み書きが不慣れな外国人配達員向けの多言語UI・音声入力を備えたアプリを提供する。
4. **管理コストの大幅削減**: 管理者がリアルタイムで全配達員の現在地と進捗を監視し、緊急時のSOS対応や、印刷順路帳に代わるスマートな管理基盤を提供する。

## 主な特徴 (Features)

### モバイル配達員アプリ (Mobile App)
高齢者や外国人留学生など、多様な配達員の働き方に配慮したユニバーサルデザインを採用しています。
- デジタル順路帳 & 地図ナビゲーション: 紙の順路帳を廃止し、GPSと連動したリアルタイムナビ。
- 多言語サポート: 日本語・英語・ベトナム語・中国語・韓国語・ネパール語の6言語に対応。
- オフライン対応: 通信環境の悪い団地や地下でも、事前キャッシュとIndexedDBにより停止することなく配達業務を継続可能。
- 夜間モード自動切替: 早朝（3:00 - 6:00）に合わせて目に優しいダークモードと地図スタイルを自動適用。
- SOS緊急機能: トラブル時にワンタップで管理者に位置情報を送信し救助を要請。

### 管理者デスクトップダッシュボード (Admin Dashboard)
販売店のあらゆる業務を一元管理し、データに基づいた意思決定を支援します。
- AIルート最適化: 購読者の新規追加や留守止め発生時に、最も効率的な配達ルートをAIが自動再計算。
- リアルタイム配達状況: マップ上で全配達員の現在地と進捗 (% / 件数) をライブ追跡。
- ドラッグ＆ドロップ編集: ルートや順序の変更を直感的かつ即座に反映。
- 留守止め・新規一括管理: カレンダービューと連携した顧客のステータス管理機能。
- PDF印刷連携: システム障害時や予備用に従来型の「印刷順路帳」も即時出力可能。

## 技術スタック (Tech Stack)

### フロントエンド (Frontend)
- コア: React (v18) / TypeScript / Vite (v6) / React Router (v7)
- スタイリング: Tailwind CSS (v4) / Radix UI / shadcn/ui パターン
- アイコン/UI: Lucide React / Material UI (MUI v7)
- 機能拡張:
  - 地図: React Leaflet
  - チャート: Recharts
  - アニメーション: Framer Motion
  - DnD: React DnD
  - その他: Sonner (Toast), Vaul (Drawer), Embla Carousel

## 開発環境のセットアップ (Getting Started)

### 前提条件
- Node.js (v18以上推奨)
- npm または pnpm

### インストール手順

1. リポジトリのクローン（フロントエンドへの移動）
```bash
cd frontend
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーの起動
```bash
npm run dev
```
通常、http://localhost:5173 で開発サーバーが立ち上がります。
